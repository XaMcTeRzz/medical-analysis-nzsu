# 🌐 Веб-додаток для обробки медичних звітів (HTML/JavaScript)

## 🎯 Огляд

Створено сучасний веб-додаток на чистому HTML, JavaScript та CSS, який працює на будь-якому комп'ютері без встановлення додаткових програм. Додаток має красивий інтерфейс та повний функціонал для обробки медичних звітів.

## ✨ Переваги

- 🚀 **Працює скрізь** - не потребує встановлення Python або інших програм
- 📱 **Responsive дизайн** - адаптивний для будь-яких пристроїв
- 🎨 **Сучасний інтерфейс** - красивий дизайн з анімаціями
- 📊 **Інтерактивні графіки** - візуалізація даних за допомогою Chart.js
- 💾 **Експорт в Excel** - можливість завантаження результатів
- 🔍 **Пошук та фільтрація** - зручний пошук лікарів

## 🚀 Як користуватися

### Спосіб 1: Відкриття файлу
1. Відкрийте файл `index.html` у будь-якому браузері
2. Перетягніть файли Excel у зону завантаження
3. Натисніть "Обробити дані"
4. Насолоджуйтесь результатами!

### Спосіб 2: Локальний сервер
```bash
# Відкрийте папку зі звітами у терміналі
cd "c:\Users\PC\Downloads\Звіт"

# Запустіть локальний сервер (Python 3)
python -m http.server 8000

# Або використайте Node.js
npx http-server

# Відкрийте у браузері
http://localhost:8000
```

## 🎨 Інтерфейс додатку

### 📤 Завантаження файлів
- Drag & drop інтерфейс
- Візуальна індикація завантажених файлів
- Автоматична перевірка наявності необхідних файлів

### 📊 Основні функції

#### 1. Загальний звіт
- Загальна статистика (лікарі, пацієнти, суми)
- Детальна таблиця з сортуванням
- Пошук лікарів в реальному часі

#### 2. Графіки
- Топ-10 лікарів за сумою (горизонтальна діаграма)
- Топ-10 лікарів за кількістю пацієнтів
- Кругова діаграма розподілу сум
- Scatter plot залежностей

#### 3. Деталі по лікарях
- Вибір конкретного лікаря з dropdown
- Індивідуальна статистика лікаря
- Список пацієнтів з сумами
- Графік розподілу сум пацієнтів

#### 4. Завантаження
- Експорт результатів в Excel
- Статистичні показники
- Зведені таблиці

3. **Читання** - парсинг Excel файлів у JavaScript

## 🐛 Усунення несправностей

### Поширені проблеми

1. **Файли не завантажуються**
   - Перевірте формати файлів (.xlsx, .xls)
   - Переконайтеся, що файли не пошкоджені

2. **Графіки не відображаються**
   - Перевірте підключення до інтернету (для CDN)
   - Оновіть сторінку

3. **Експорт не працює**
   - Перевірте, чи оброблено дані
   - Спробуйте інший браузер

# Програма для обробки медичних звітів

## Призначення

Програма автоматично обробляє медичні звіти з Excel файлів та створює звіти по кожному лікарю з інформацією про:
- Кількість унікальних пацієнтів
- Загальну суму до сплати
- Детальну інформацію по кожному пацієнту

## Алгоритм роботи

1. **Читання даних**: Програма читає два Excel файли:
   - `Січень повна.xlsx` - повний звіт з усіма медичними записами
   - `Звіт_по_включеним_тижням_по_пацієнтам_СІЧЕНЬ.xlsx` - звіт з сумами до сплати

2. **Фільтрація по лікарях**: З повного файлу витягуються унікальні лікарі (ПІБ)

3. **Зіставлення даних**: Для кожного лікаря:
   - Знаходяться всі його унікальні пацієнти (по ID)
   - Підставляються дані у файл звіту
   - Рахується загальна кількість пацієнтів та сума

4. **Створення звіту**: Генерується новий Excel файл з:
   - Загальним звітом по всіх лікарях
   - Окремими аркушами з детальною інформацією по кожному лікарю

## Встановлення

Програма потребує встановлення Python та бібліотек:

```bash
pip install pandas openpyxl
```

## Запуск

### Спосіб 1: Через головний скрипт
```bash
python run_report.py
```

### Спосіб 2: Безпосередньо
```bash
python medical_report_processor_by_doctor.py
```

## Вхідні файли

1. **Січень повна.xlsx** - повний медичний звіт
   - Містить інформацію про медичні записи
   - Стовпці: Унікальний код пацієнта, Медичний працівник, Посада, тощо

2. **Звіт_по_включеним_тижням_по_пацієнтам_СІЧЕНЬ.xlsx** - фінансовий звіт
   - Містить інформацію про суми до сплати
   - Стовпці: ID пацієнта, До сплати, грн, тощо

## Вихідні файли

Програма створює Excel файл з назвою формату:
`Звіт_по_лікарях_ПІБ_ГГГГММДД_ГГХХХ.xlsx`

Файл містить:
- **Аркуш "Загальний звіт"** - зведена таблиця по всіх лікарях
- **Аркуші з ПІБ лікарів** - детальна інформація по кожному пацієнту

## Структура вихідного звіту

### Загальний звіт
| ПІБ лікаря | Посада | Кількість пацієнтів | Загальна сума до сплати (грн) |
|------------|--------|-------------------|------------------------------|
| Іваненко І.І. | Лікар-терапевт | 45 | 123456.78 |

### Детальний звіт по лікарю
| ID пацієнта | Сума до сплати |
|-------------|----------------|
| ABC123... | 1234.56 |
| DEF456... | 2345.67 |

## Особливості

- Програма автоматично визначає структуру Excel файлів
- Працює з різними назвами стовпців
- Фільтрує дані по унікальних ID пацієнтів
- Створює зручні для аналізу звіти
- Обробляє великі обсяги даних

## Увага

- Переконайтеся, що обидва вхідні файли знаходяться в тій самій папці
- Файли повинні мати очікувану структуру (див. розділ "Вхідні файли")
- Програма створює нові файли, не змінюючи вихідні

## Технічна підтримка

Якщо виникають проблеми:
1. Перевірте наявність та назви вхідних файлів
2. Перевірте встановлення бібліотек: `pip install pandas openpyxl`
3. Переконайтеся, що файли не відкриті в інших програмах
